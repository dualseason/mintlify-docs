title: "Imagen 4.0 XCompute 图像生成"
description: "- 异步处理模式，返回任务ID用于后续查询"

- 基于 Google Imagen 4 模型，高质量图像生成
- 仅支持文生图，不支持图生图 / 参考图
- 支持比例：1:1（正方形）、4:3（横屏）、3:4（竖屏）、16:9（横屏）、9:16（竖屏），默认 16:9
- 生成的图像链接，有效期为24小时，请尽快保存

## Authorizations

  获取 API Key：

  访问 [API Key 管理页面](https://XCompute.us/console/token) 获取您的 API Key

  Authorization: Bearer YOUR_API_KEY

## Body

<ParamField body="model" type="string" default="imagen-4.0-apimart" required>
  图像生成模型名称

  固定填写 `imagen-4.0-apimart`
</ParamField>

<ParamField body="prompt" type="string" required>
  图像生成的文本描述

  英文 / 中文描述均可
</ParamField>

<ParamField body="n" type="integer" default="1">
  生成图片张数

  取值范围：1
</ParamField>

<ParamField body="size" type="string" default="16:9">
  图像生成的比例

  支持的格式：

  * `1:1` - 正方形
  * `4:3` - 横屏
  * `3:4` - 竖屏
  * `16:9` - 横屏（默认）
  * `9:16` - 竖屏

  <Note>
    传入不在支持列表里的比例（如 `21:9`）会被自动回退到默认 `16:9`，不会报错。
  </Note>
</ParamField>

<Note>
  不支持的字段：`image`、`image_urls`、`mask`、`quality` 等会被忽略。如果传入 `image_urls` / `image` 字段会直接报错：`google_imagen4 currently supports text-to-image only`。
</Note>
