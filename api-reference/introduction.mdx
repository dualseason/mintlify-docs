## 使用方式

1. 获取你的 API Key：`https://XCompute.us/console/token`
2. 将 Base URL 设置为 `https://XCompute.us/v1`
3. 选择目标模型并发起请求
4. 对图像和视频接口使用任务查询接口轮询结果

## 参考示例

```python theme={null}
from openai import OpenAI

client = OpenAI(
    base_url="https://XCompute.us/v1",
    api_key="your-apimart-key",
)

response = client.chat.completions.create(
    model="gpt-5",
    messages=[
        {"role": "user", "content": "你好，介绍一下 XCompute"}
    ],
)

print(response)
```

<Note>
  图像和视频接口大多是异步任务模式。提交后会返回 `task_id`，然后通过
  [任务状态接口](/api-reference/tasks/status) 继续查询。
</Note>
