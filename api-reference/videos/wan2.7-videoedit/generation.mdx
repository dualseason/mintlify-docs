- 支持保留原视频时长或自定义输出时长

## 请求参数

<ParamField body="model" type="string" required>
  视频生成模型名称，固定为 `wan2.7-videoedit`
</ParamField>

<ParamField body="video_urls" type="array<string>" required>
  待编辑的原始视频 URL 数组

  <Warning>
    **只取第 1 个视频**
  </Warning>

  <Note>
    **视频限制：**

    * 格式：mp4、mov
    * 时长：2～10s
    * 分辨率：宽度和高度范围为 \[240, 4096] 像素
    * 宽高比：1:8 \~ 8:1
    * 文件大小：不超过 100MB
  </Note>
</ParamField>

<ParamField body="prompt" type="string">
  编辑指令，描述希望对视频做什么改变，最多 5000 字符

  <Note>
    不传时模型将进行默认的风格迁移
  </Note>

  示例：`"将人物的服装改为红色礼服"`、`"将背景替换为雪山场景"`
</ParamField>

<ParamField body="negative_prompt" type="string">
  反向提示词，描述不希望出现的内容，最多 500 字符
</ParamField>

<ParamField body="image_urls" type="array<string>">
  参考图片 URL 数组，最多 4 张

  用于指定目标风格或外观（如风格迁移的目标样式参考）
</ParamField>

<ParamField body="resolution" type="string" default="1080P">
  输出视频分辨率

  可选值：

  * `720P` - 标清
  * `1080P` - 高清（默认）
</ParamField>

<ParamField body="duration" type="integer" default="0">
  输出视频时长（秒）

  * `0`（默认）：保留原视频完整时长
  * `2-10` 的整数：从头截取指定时长

  <Note>
    当 `duration=0` 时，以输出视频实际时长计费

    指定时长不能超过 `video_urls` 原始视频的时长
  </Note>
</ParamField>

<ParamField body="size" type="string">
  输出画面宽高比

  支持的格式：

  * `16:9` - 横版宽屏
  * `9:16` - 竖版长屏
  * `1:1` - 正方形
  * `4:3` - 横版
  * `3:4` - 竖版

  <Note>
    不传则与输入视频宽高比保持一致
  </Note>
</ParamField>

<ParamField body="prompt_extend" type="boolean" default="true">
  是否开启 prompt 智能改写

  对较短提示词效果提升明显，但会增加耗时

  默认值：`true`
</ParamField>

<ParamField body="watermark" type="boolean" default="false">
  是否在生成的视频上添加 "AI生成" 水印

  * `true`：添加水印
  * `false`：不添加水印（默认）
</ParamField>

<ParamField body="seed" type="integer">
  种子整数，用于控制生成内容的随机性

  取值范围：`≥0` 的整数

  <Note>
    * 相同的请求下，模型收到不同的 seed 值（如：不指定 seed 值），将生成不同的结果
    * 相同的请求下，模型收到相同的 seed 值，会生成类似的结果，但不保证完全一致
  </Note>
</ParamField>

<ParamField body="metadata" type="object">
  附加参数对象

  <Expandable title="metadata 字段">
    <ParamField body="audio_setting" type="string" default="auto">
      音频处理方式：

      * `auto`（默认）：AI 根据编辑后的视频内容自动重新生成匹配音频
      * `origin`：强制保留原视频音频，适合有重要背景音/对话的视频
    </ParamField>
  </Expandable>
</ParamField>

## 使用场景

### 场景 1：基础视频编辑（最简）

```json theme={null}
{
  "model": "wan2.7-videoedit",
  "video_urls": ["https://cdn.example.com/original.mp4"],
  "prompt": "将背景替换为雪山场景"
}
```

### 场景 2：风格迁移（带参考图）

```json theme={null}
{
  "model": "wan2.7-videoedit",
  "prompt": "将视频风格迁移为参考图的动漫风格",
  "video_urls": ["https://cdn.example.com/original.mp4"],
  "image_urls": [
    "https://cdn.example.com/anime_style.jpg"
  ],
  "resolution": "1080P",
  "watermark": false
}
```

### 场景 3：保留原视频音频

适合有重要背景音或人物对话的视频：

```json theme={null}
{
  "model": "wan2.7-videoedit",
  "video_urls": ["https://cdn.example.com/speech.mp4"],
  "prompt": "将背景替换为山间小路",
  "metadata": { "audio_setting": "origin" }
}
```

### 场景 4：完整参数

```json theme={null}
{
  "model": "wan2.7-videoedit",
  "prompt": "将人物的服装改为红色礼服",
  "negative_prompt": "模糊、失真",
  "video_urls": ["https://cdn.example.com/original.mp4"],
  "image_urls": ["https://cdn.example.com/reference.jpg"],
  "resolution": "1080P",
  "duration": 0,
  "size": "16:9",
  "prompt_extend": true,
  "watermark": false,
  "seed": 888,
  "metadata": {
    "audio_setting": "origin"
  }
}
```

## 音频处理说明

| audio\_setting | 说明                    | 适用场景              |
| -------------- | --------------------- | ----------------- |
| `auto`（默认）     | AI 根据编辑后的视频内容重新生成匹配音频 | 视觉风格大幅改变，希望音频同步更新 |
| `origin`       | 强制保留原视频音轨             | 有重要背景音乐、人物对话的视频   |

<Note>
  **查询任务结果**

  视频生成为异步任务，提交后会返回 `task_id`。使用 [获取任务状态](/api-reference/tasks/status) 接口查询生成进度和结果。
</Note>
